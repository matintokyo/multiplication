
# Sprint de Multiplications

Jeu mobile simple pour s'entraîner aux tables de multiplication.

Comment ça marche :
- Choisissez les tables de multiplication à pratiquer (2–9).
- Le jeu génère 20 opérations aléatoires parmi les tables sélectionnées.
- Répondez aux 20 opérations avec le pavé numérique à l'écran en moins de 60 secondes pour gagner.
