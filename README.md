
# Multiplication Sprint

Jeu mobile simple pour s'entraîner aux tables de multiplication (HTML/CSS/JS). Ouvrez `index.html` dans un navigateur ou hébergez sur GitHub Pages.

Comment ça marche :
- Choisissez les tables de multiplication à pratiquer (2–9).
- Le jeu génère 20 opérations aléatoires parmi les tables sélectionnées.
- Répondez aux 20 opérations avec le pavé numérique à l'écran en moins de 60 secondes pour gagner.

Publication sur GitHub Pages :
1. Créez un nouveau dépôt et poussez ces fichiers à la racine du dépôt.
2. Dans les paramètres du dépôt → Pages, définissez la source sur la branche `main` (root).
3. Visitez l'URL fournie par GitHub Pages.
